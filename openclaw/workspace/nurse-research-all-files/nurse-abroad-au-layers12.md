## Layer 1 - 官方政策深潜

### 1.1 澳洲移民局官方政策 (immi.homeaffairs.gov.au)

#### 1.1.1 482 Skills in Demand Visa (技能需求签证)

**官方链接**: 
- https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-listing/skills-in-demand-visa-subclass-482
- https://immi.homeaffairs.gov.au/visas/employing-and-sponsoring-someone/sponsoring-workers

**政策要点**:

482签证于2024年12月7日正式取代原有的482 TSS签证，是澳洲雇主担保工作的主要途径。

**核心要求**:
- **职业清单**: 申请人职业必须在Core Skills Occupation List (CSOL)核心技能职业清单上
- **护士职业**: Registered Nurse (注册护士) 明确在CSOL清单中
- **工作经验**: 至少1年相关工作经验（全职或等同兼职）
- **英语要求**: 雅思总分5.0（单项不低于5.0）或同等水平
- **年龄要求**: 无严格年龄限制（但45岁以上转永居可能有难度）
- **技能评估**: 部分职业需要通过相关技能评估机构评估
- **雇主担保**: 需要获得澳洲合格雇主的担保

**签证结构**:
| 类别 | 适用人群 | 有效期 |
|------|----------|--------|
| Core Skills Pathway | 核心技能职业清单上的申请人 | 最长4年 |
| Specialist Skills Pathway | 高薪专业人士（年薪>13.5万澳元） | 最长4年 |
| Essential Skills Pathway | 低薪但关键行业工人 | 待定 |

**482签证官方原文摘录**:
> "The Skills in Demand visa is a temporary visa that enables employers to address labour shortages by bringing in skilled workers where employers cannot source an appropriately skilled Australian worker."

#### 1.1.2 189 Skilled Independent Visa (独立技术移民)

**官方链接**:
- https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-listing/skilled-independent-189

**政策要点**:

189签证是澳洲技术移民的"黄金签证"，一步到位获得永久居留权，无需州担保或雇主担保。

**核心要求**:
- **职业清单**: 必须在MLTSSL (Medium and Long-term Strategic Skills List)中长期战略技能清单上
- **护士职业**: Registered Nurse (ANZSCO 2544) 在MLTSSL清单上
- **EOI打分**: 至少65分（根据邀请轮次，实际获邀分数通常更高）
- **技能评估**: 必须通过相关职业评估机构（护士为ANMAC或AHPRA）
- **英语要求**: 至少Competent English（雅思4个6或等同）
- **年龄**: 45岁以下

**EOI打分系统**:

| 项目 | 分数 |
|------|------|
| 年龄18-24岁 | 25分 |
| 年龄25-32岁 | 30分 |
| 年龄33-39岁 | 25分 |
| 年龄40-44岁 | 15分 |
| 英语 proficient (雅思4个7) | 10分 |
| 英语 superior (雅思4个8) | 20分 |
| 澳洲工作经验1-3年 | 5分 |
| 澳洲工作经验3-5年 | 10分 |
| 澳洲工作经验5-8年 | 15分 |
| 澳洲工作经验8年以上 | 20分 |
| 海外工作经验3-5年 | 5分 |
| 海外工作经验5-8年 | 10分 |
| 海外工作经验8年以上 | 15分 |
| 澳洲学历 | 5-20分 |
| 职业年(Professional Year) | 5分 |
| 社区语言(CCL/NAATI) | 5分 |
| 偏远地区学习 | 5分 |
| 州/领地担保(190) | 5分 |
| 单身(无配偶) | 10分 |
| 配偶有技能评估+英语 | 10分 |
| 配偶仅有英语 | 5分 |

**189官方原文摘录**:
> "This visa lets invited workers with skills we need, live and work permanently anywhere in Australia."

#### 1.1.3 190 Skilled Nominated Visa (州担保技术移民)

**官方链接**:
- https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-listing/skilled-nominated-190

**政策要点**:

190签证是州/领地政府担保的永居签证，获得担保后可额外加5分。

**核心要求**:
- **职业清单**: 必须在各州担保职业清单上（各州清单不同）
- **州担保**: 需要获得某一州/领地政府的担保
- **居住承诺**: 获得签证后需在担保州居住至少2年
- **EOI打分**: 至少65分（含州担保5分）
- **其他要求**: 与189签证基本相同

**各州担保政策差异对比**:

| 州/领地 | 护士职业担保情况 | 特殊要求 | 邀请频率 |
|---------|------------------|----------|----------|
| **新州 (NSW)** | 定期开放，竞争激烈 | 高分优先，有居住要求 | 每月 |
| **维州 (VIC)** | 护士需求量大 | 需在维州居住或工作 | 每月 |
| **昆州 (QLD)** | 分境内/境外通道 | 境内申请人优先 | 每月 |
| **西澳 (WA)** | 护士属于优先职业 | 建筑/医疗行业优先 | 每月 |
| **南澳 (SA)** | 护士长期紧缺 | 优先考虑本地毕业生 | 每月 |
| **塔州 (TAS)** | 护士需求稳定 | 优先考虑塔州毕业生 | 每月 |
| **首都领地 (ACT)** | 有独立打分矩阵 | 需Matrix打分 | 每月 |
| **北领地 (NT)** | 护士紧缺程度高 | 优先考虑本地毕业生 | 滚动邀请 |

**190官方原文摘录**:
> "This visa lets nominated skilled workers live and work in Australia as permanent residents."

#### 1.1.4 491 Skilled Work Regional Provisional Visa (偏远地区技术移民)

**官方链接**:
- https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-listing/skilled-work-regional-provisional-491

**政策要点**:

491是临时签证（5年），在偏远地区居住工作满3年后可转191永居签证。

**偏远地区定义**:
- **Category 2**: 珀斯、阿德莱德、黄金海岸、阳光海岸、堪培拉等
- **Category 3**: 其他所有地区和城市

**优势**:
- EOI额外加15分
- 部分州担保要求相对宽松
- 可转191永居

#### 1.2.2 IQNM评估流程 (Internationally Qualified Nurses and Midwives)

**官方链接**:
- https://www.ahpra.gov.au/registration/registration-process/overseas-qualified-practitioners/internationally-qualified-nurses-midwives.aspx

**IQNM评估框架三大阶段**:

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  1. Self-check  │ → │ 2. Portfolio    │ → │ 3. OBA Exam     │
│   (自我评估)     │    │  (档案评估)      │    │  (OBA考试)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

**阶段1: Self-check (自我评估)**

**目的**: 确认申请人是否符合IQNM路径的基本资格

**检查内容**:
- 学历是否符合澳洲护士教育标准
- 是否持有有效的海外护士注册/执照
- 是否有足够的临床经验
- 英语能力是否达标

**官方链接**: https://www.ahpra.gov.au/registration/registration-process/overseas-qualified-practitioners/iqnm-self-check.aspx

**阶段2: Portfolio Assessment (档案评估)**

**官方链接**: https://www.ahpra.gov.au/News/2021-09-30-IQNM-portfolio-assessment.aspx

**所需文件清单**:
| 文件类型 | 要求 |
|----------|------|
| 身份证明 | 护照、出生证明 |
| 学历证明 | 护理学位证书、成绩单（需翻译） |
| 注册证明 | 原国家护士注册证书/执照 |
| 工作经历 | 雇主推荐信、工作证明 |
| 无犯罪证明 | 所有居住国家的无犯罪记录 |
| 英语成绩 | 雅思/OET成绩 |

**评估结果**:
- **符合标准**: 直接进入OBA考试阶段
- **需要补充**: 要求提供额外文件
- **不符合**: 可能需要先完成桥梁课程

**阶段3: OBA (Outcomes-Based Assessment)**

**官方链接**: https://www.ahpra.gov.au/support/iqnm-outcomes-based-assessment.aspx

**OBA官方原文摘录**:
> "The Outcomes-Based Assessment (OBA) is a two-stage assessment process for internationally qualified nurses and midwives seeking registration in Australia."

---

#### 1.2.3 OBA考试详解

**OBA考试结构**:

```
                    OBA考试
                      │
        ┌─────────────┴─────────────┐
        │                           │
   ┌────▼────┐                 ┌────▼────┐
   │   NCLEX │                 │  OSCE   │
   │   (MCQ) │                 │(实操)   │
   │ $350 USD│                 │$4,000   │
   │  机考   │                 │ 面考    │
   └────┬────┘                 └────┬────┘
        │                           │
        ▼                           ▼
   通过 → 进入OSCE            通过 → 获得注册
   未通过 → 重考              未通过 → 重考
```

**MCQ考试 (Multiple Choice Question)**

- **费用**: $200 USD考试费 + $150 USD国际考点费 = **$350 USD** [官方AHPRA 2024]
- **形式**: 计算机自适应考试，可在Pearson VUE全球考点进行
- **题量**: 约75-145题（根据表现动态调整）
- **时长**: 最多5小时
- **通过标准**: 基于题目难度和正确率综合评分
- **考试内容**: 基于NCLEX-RN考试框架
  - Safe and Effective Care Environment
  - Health Promotion and Maintenance
  - Psychosocial Integrity
  - Physiological Integrity

**MCQ重考规则**:
- 失败后需等待45天方可重考
- 最多可重考3次
- 每次重考需重新支付$350 USD

**OSCE考试 (Objective Structured Clinical Examination)**

- **费用**: **$4,000 AUD** [官方NMBA 2024]
- **地点**: 澳洲阿德莱德NCAS (National Centre for Assessment of Skills)中心
- **形式**: 实操临床技能考试
- **考位**: 每场次有限，需提前预约
- **内容**: 模拟真实临床场景，评估护士的实际操作能力
- **时长**: 约4-6小时

**OSCE考试内容模块**:
| 模块 | 内容 |
|------|------|
| 健康评估 | 体格检查、病史采集 |
| 药物管理 | 给药、注射、输液 |
| 伤口护理 | 换药、清创、包扎 |
| 急症处理 | CPR、急救技能 |
| 沟通技能 | 患者教育、团队合作 |
| 文档记录 | 护理记录、交接班 |

**OSCE重考规则**:
- 可针对未通过的部分单独重考
- 重考费用: **$2,000 AUD**（部分重考）[论坛用户分享 2024]
- 考位紧张，通常需等待2-6个月

#### 1.2.4 英语要求

**官方链接**: https://www.nursingmidwiferyboard.gov.au/Registration-Standards/English-language-skills.aspx

**接受的语言考试**:

| 考试 | 要求分数 | 备注 |
|------|----------|------|
| **雅思 (IELTS)** | 总分7.0，单项不低于7.0 | 最常用 |
| **OET** | 单项不低于B | 医疗英语考试 |
| **PTE Academic** | 总分65，单项不低于65 | 机考 |
| **TOEFL iBT** | 总分94，听/读24，写27，说23 | 较少使用 |

**官方原文摘录**:
> "IELTS: A minimum overall band score of 7.0 with no individual band less than 7.0"

**重要提示**: AHPRA接受雅思单项重考（One Skill Retake），即如果某一项未达标，可以只重考那一项。

## Layer 2 - 认证流程深潜

### 2.1 OBA路径详细步骤（中国护士主要路径）

```
┌─────────────────────────────────────────────────────────────────────┐
│                    中国护士OBA注册完整流程                            │
└─────────────────────────────────────────────────────────────────────┘

步骤1: 学历与工作经验评估
├─ 确认国内护理学历（大专/本科）
├─ 确认国内护士执照有效
├─ 确认至少3个月临床工作经验
└─ 时间: 1-2周

步骤2: 英语考试准备与考试
├─ 选择雅思或OET
├─ 备考（建议3-6个月）
├─ 参加考试
└─ 时间: 3-6个月 | 费用: ¥2,000-3,000/次

步骤3: IQNM Self-check
├─ 在线完成自我评估问卷
├─ 确认符合OBA路径资格
└─ 时间: 即时 | 费用: 免费
       ↓
步骤4: 创建AHPRA账户
├─ 注册AHPRA在线账户
├─ 选择IQNM评估服务
└─ 时间: 1天 | 费用: 免费
       ↓
步骤5: Portfolio评估申请
├─ 支付评估费
├─ 上传所有所需文件
├─ 等待评估结果
└─ 时间: 6-12周 | 费用: $635 AUD
       ↓
步骤6: 收到OBA邀请
├─ 确认进入OBA考试阶段
├─ 收到考试安排通知
└─ 时间: 随评估结果发出
       ↓
步骤7: MCQ考试预约
├─ 在Pearson VUE注册
├─ 选择考试地点（中国或海外）
├─ 支付考试费
└─ 时间: 1-2周预约 | 费用: $350 USD
       ↓
步骤8: MCQ考试
├─ 参加考试
├─ 等待成绩（通常48小时）
└─ 时间: 1天 | 通过 → 进入OSCE
                未通过 → 45天后重考
       ↓
步骤9: OSCE考试预约
├─ 联系NCAS预约考位
├─ 安排赴澳洲行程
├─ 支付考试费
└─ 时间: 2-6个月等待考位 | 费用: $4,000 AUD
       ↓
步骤10: OSCE考试与注册
├─ 前往阿德莱德参加考试
├─ 等待成绩（通常4-6周）
├─ 通过 → 获得注册资格
└─ 时间: 1-2周 | 获得RN注册！
```
