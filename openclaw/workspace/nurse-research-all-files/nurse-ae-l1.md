# 阿联酋护士出国 - Layer 1: 官方政策深潜

> 调研日期: 2026年2月11日  
> 调研范围: 阿联酋卫生部官方政策（MOHAP, DHA, DOH）、护士注册政策、合同与工作环境

## 一、阿联酋三大卫生部门概述

### 1.1 联邦与地方卫生部门分工

阿联酋的医疗监管体系采用**联邦制与酋长国自治相结合**的模式：

| 机构 | 管辖范围 | 职责 |
|------|----------|------|
| **MOHAP** (Ministry of Health and Prevention) | 联邦层面 + 非自由区酋长国 | 监管沙迦、阿治曼、乌姆盖万、哈伊马角、富查伊拉等酋长国 |
| **DHA** (Dubai Health Authority) | 迪拜酋长国 | 监管迪拜全境医疗机构，包括迪拜医疗城(DHCC) |
| **DOH** (Department of Health - Abu Dhabi) | 阿布扎比酋长国 | 监管阿布扎比全境医疗机构 |

### 1.2 PQR统一标准

三个机构共同采用 **Unified Healthcare Professional Qualification Requirements (PQR)** 第三版，这是阿联酋医疗专业人员资格要求的统一标准文件，确保全国医疗服务质量的一致性。

- **核心要素**: 教育标准、临床经验、执业许可要求
- **更新日期**: 2024-2025年持续更新
- **下载链接**: [Unified Healthcare Professional Qualification Requirements](https://www.doh.gov.ae/-/media/D112FF75EF0140439A733B16081292DF.ashx)

## 二、护士注册官方政策详解

### 2.1 注册流程总览

所有三个机构的注册流程基本相似，遵循以下步骤：

```
┌─────────────────────────────────────────────────────────────┐
│                    阿联酋护士注册流程                         │
├─────────────────────────────────────────────────────────────┤
│ 1. 自评评估 (Self-Assessment)                                 │
│    └─ 在Sheryan/Malaffi系统使用PQR自评工具                   │
│                                                               │
│ 2. DataFlow学历验证 (Primary Source Verification)            │
│    └─ 提交学历、执照、工作经验证明进行原始来源验证             │
│                                                               │
│ 3. 提交正式申请                                               │
│    └─ 上传所有文件，支付申请费用                              │
│                                                               │
│ 4. 获取CBT资格ID (Computer-Based Test Eligibility ID)       │
│    └─ DHA/DOH/MOHAP审核后发放考试资格                        │
│                                                               │
│ 5. 参加Prometric考试                                          │
│    └─ 在全球8,000个考试中心参加考试                          │
│                                                               │
│ 6. 获取评估证书/执业资格                                      │
│    └─ 通过后获得执业资格，雇主激活执照                        │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 DHA (迪拜) 详细要求

#### 申请平台
- **Sheryan Portal**: https://services.dha.gov.ae/sheryan/
- 需创建个人账户并提交"Professional Registration"申请

#### 必需文件
1. 学历证明（护理学学位/文凭）
2. 学术成绩单
3. 本国执业执照
4. 工作经验证明（通常要求2年以上临床经验）
5. Good Standing Certificate（品行证明）
6. 有效护照复印件

#### 考试要求
- **考试形式**: 计算机化考试 (CBT)
- **考试管理**: Prometric（全球160个国家8,000+考试中心）
- **考试时长**: 约2小时
- **题目数量**: 100-150题
- **及格分数**: 60%-70%（根据具体考试类型）
- **考试语言**: 仅英语
- **考试费用**: 约 **USD 220-280**（直接支付给Prometric）
- **尝试次数**: 最多3次

#### DataFlow验证费用
- **护士/辅助医疗**: **935 AED**（约USD 255）
- **医生**: **1,235 AED**（约USD 336）
- **处理时间**: 15-24个工作日

### 2.3 DOH (阿布扎比) 详细要求

#### 申请平台
- **TAMM/DOH系统**: https://www.doh.gov.ae/
- PQR文档下载: https://www.doh.gov.ae/en/pqr

#### 特点
- 与DHA使用统一的PQR标准
- 接受从DHA转移的DataFlow报告
- 学历评估费用: **护士400 AED**

### 2.4 MOHAP (联邦/其他酋长国) 详细要求

#### 申请平台
- **MOHAP官网**: https://mohap.gov.ae/en/w/evaluation-of-health-professional
- **MOHAP智能App**

#### 服务费用
| 职业类别 | 费用 (AED) |
|----------|------------|
| 医生/牙医 | 500 |
| 护士/助产士 | **400** |
| 传统/补充医学 | 300 |
| 辅助医疗 | 300 |

#### 申请条件
- 职业实践中断不得超过3年
- 需持有原籍国认可证书
- 需通过所有规定考试（如适用）
- 黑名单候选人禁止申请

#### 所需文件
- 学历证明（学位证书）
- 学术成绩单
- 工作经验证明（应届毕业生可选）
- 执业执照（应届毕业生可选）
- 品行证明（应届毕业生可选）
- 有效护照复印件
- 以往DataFlow报告副本

## 三、学历验证 (DataFlow) 详解

### 3.1 什么是DataFlow

DataFlow Group是阿联酋卫生部门指定的**原始来源验证 (Primary Source Verification, PSV)** 服务提供商，负责验证：
- 学历证书真实性
- 执业执照有效性
- 工作经验证明

### 3.2 DataFlow优势
- 与DHA/DOH/MOHAP系统直接对接
- 一次验证，可用于多个机构申请
- 验证报告可在系统中关联
- 全球160,000+发证机构网络覆盖200+国家

### 3.3 验证流程
1. 在线提交申请
2. 上传所需文件
3. DataFlow直接联系发证机构验证
4. 出具正式验证报告
5. 报告关联到Sheryan/MOHAP系统

## 四、合同与工作环境

### 4.1 薪资水平 (2024-2025)

#### 月薪范围 (免税 AED)

| 经验级别 | 月薪范围 (AED) | 折合人民币* |
|----------|----------------|-------------|
| 入门级 (0-2年) | 4,000 - 7,500 | ¥7,800 - ¥14,600 |
| 中级 (3-10年) | 8,000 - 14,000 | ¥15,600 - ¥27,300 |
| 注册护士 (RN) | 8,000 - 12,000 | ¥15,600 - ¥23,400 |
| 专科/高级护士 | 12,000 - 25,000+ | ¥23,400 - ¥48,800+ |
| 重症监护/急诊/新生儿 | 14,000 - 20,000 | ¥27,300 - ¥39,000 |

*按 1 AED ≈ 1.95 CNY 计算

#### 年薪参考
- **入门级**: AED 84,000 - 120,000/年
- **中级**: AED 120,000 - 168,000/年  
- **高级护士**: AED 216,000 - 300,000/年

### 4.2 典型福利包

大多数雇主提供的标准福利包括：

| 福利项目 | 说明 |
|----------|------|
| **住宿** | 免费住房或住房补贴 |
| **医疗保险** | 全面健康保险覆盖 |
| **年度机票** | 每年往返原籍国机票 |
| **年终奖金** | 服务期满酬金 (End-of-service gratuity) |
| **带薪假期** | 年度带薪休假 (通常30天) |
| **签证费用** | 雇主承担工作签证及ID卡费用 |

### 4.3 工作环境特点

#### 优势
- **免税收入**: 阿联酋无个人所得税，到手收入高
- **国际化环境**: 医疗团队来自全球各地
- **先进设施**: 迪拜/阿布扎比拥有世界级医疗设施
- **职业发展**: 医疗旅游发达，专科发展机会多
- **生活便利**: 英语普及，生活设施完善

#### 注意事项
- 文化适应：伊斯兰国家，需尊重当地习俗
- 工作强度：根据科室和医院不同而有差异
- 气候：夏季炎热（40°C+），室内工作为主

## 五、中国护士特别提醒

### 5.1 中国护士注册路径

1. **学历认证**: 中国护理学历需经DataFlow验证
2. **语言要求**: 所有考试和交流使用英语，建议雅思5.5+
3. **经验要求**: 通常需2年以上临床工作经验
4. **中国护士执照**: 需持有有效中国护士执业证书

### 5.2 已有中国护士基础
- 阿联酋已有一定数量的中国护士
- 迪拜、阿布扎比国际化程度高，对中国护士友好
- 部分医院有中文服务需求（面向中国游客/居民）

### 5.3 建议准备事项
- 提前准备学历公证和认证
- 尽早开始DataFlow验证流程（需15-24个工作日）
- 准备英语考试（如需要）
- 研究目标医院和科室

## 七、总结

### 核心要点

1. **注册流程标准化**: DHA/DOH/MOHAP采用统一的PQR标准，流程清晰透明

2. **DataFlow是关键**: 学历验证是整个流程的核心环节，建议尽早启动

3. **免税薪资吸引力高**: 迪拜/阿布扎比护士薪资免税，综合福利优厚

4. **考试机会充足**: Prometric全球考点丰富，可在中国国内完成考试

5. **医疗旅游驱动需求**: 迪拜/阿布扎比医疗旅游发达，护士需求持续旺盛

### 风险提示

- 注册流程需要3-6个月不等，需提前规划
- DataFlow验证费用较高，需准备充足资金
- 英语能力是必要条件，需提前准备
- 不同医院薪资福利差异较大，需仔细比较
