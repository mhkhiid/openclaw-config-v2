## 一、关键概念区分

### 1.1 正护士（正看護師）vs 介护福祉士

| 项目 | 正护士 | 介护福祉士 |
|------|--------|------------|
| **学历要求** | 中国护理专业3年以上大专/本科 | 无严格学历要求 |
| **资格要求** | 中国护士资格证 + 日本看护师国家考试 | 日本介护福祉士国家考试 |
| **工作内容** | 医疗护理、注射、给药等 | 生活照料、康复辅助 |
| **薪资水平** | 年薪400-600万日元 | 年薪300-400万日元 |
| **签证类型** | 医疗签证/工作签证 | 特定技能签证/工作签证 |
| **日语要求** | N1水平 | N4-N3水平 |

### 1.2 签证路径

1. **EPA护士候选**（经济伙伴关系协定）
   - 中国参与较少，主要来自印尼、菲律宾
   - 需在日本接收机构工作4-6年
   
2. **特定技能签证（介护）**
   - 2019年新设
   - 考试通过后可无限期更新
   - 可携家属，10年后可申请永住

3. **工作签证（技术·人文知识·国际业务）**
   - 适用于正护士
   - 需日本医疗机构直接聘用

## 三、失败/放弃案例分析

**基本信息**
- 平台：知乎匿名回答
- 发布时间：2022年

**背景**
- 国内护士，工作5年
- 2019年赴日读语言学校

**失败经过**
| 时间 | 事件 |
|------|------|
| 2019.04 | 赴日读语言学校 |

**失败原因分析**

2. **经济压力**：语言学校费用+生活费耗尽积蓄
3. **疫情影响**：2020-2021年打工机会减少
4. **心理落差**：国内是资深护士，在日本成"学生"

**经验教训**
- 建议在国内至少通过N2再赴日
- 准备充足的备用资金（至少2年生活费）
- 做好心理建设，接受身份落差

## 四、EPA护士候选项目（中国参与较少）

### 项目概述
- **EPA**：Economic Partnership Agreement
- **参与国**：印尼、菲律宾、越南为主
- **中国参与**：极少，主要通过其他路径

### 项目流程
1. 母国参加护士资格考试
2. 赴日后进入日本医疗机构
3. 工作期间学习日语
4. 3-4年内通过日本看护师考试

### 中国护士的替代方案
由于中国参与EPA项目较少，中国护士主要通过以下路径：
- 自费留学→语言学校→看护师考试
- 直接找工作签证（需日语N1+资格认定）

## 六、综合建议与总结

### 6.1 路径选择建议

| 条件 | 推荐路径 | 预计费用 | 预计时间 |
|------|----------|----------|----------|
| 护士本科+日语N1 | 直接资格认定+考试 | 3-5万 | 6-12个月 |
| 护士大专+日语零基础 | 语言学校→看护师考试 | 20-30万 | 2-3年 |
| 非护理专业 | 特定技能介护 | 3-5万 | 6-12个月 |
| 经济有限+急赴日 | 特定技能介护→转行 | 3-5万 | 6个月+ |

### 6.2 关键成功因素

1. **日语能力**：越早准备越好，建议国内至少达到N2
2. **经济基础**：准备25-30万预算较安全
3. **心理建设**：接受从"资深护士"到"学生/新人"的身份转变
4. **信息搜集**：多平台了解真实案例，警惕营销号

### 6.3 风险提示

1. **时间风险**：整个流程可能长达2-3年
2. **经济风险**：语言学校阶段无收入，持续支出
3. **考试风险**：看护师考试通过率约80-90%，但有失败可能
4. **中介风险**：行业内存在不规范中介，需谨慎选择

### 6.4 可信度标记说明

| 标记 | 含义 |
|------|------|
| [DIY成功] | 个人独立完成，可信度较高 |
| [中介协助] | 通过中介完成，需甄别中介质量 |
| [可疑营销] | 内容可能有营销成分，需谨慎参考 |
| [失败放弃] | 失败案例，具有重要的参考价值 |
