## 一、重要背景信息

### 香港护士考试基本规则
| 项目 | 详情 |
|------|------|
| 考试名称 | HKRN (Hong Kong Registered Nurse) |
| 监管机构 | 香港护士管理局 (Nursing Council of Hong Kong) |
| 考试机会 | 每年2次，每人仅**3次考试机会** |
| 考试地点 | 仅限香港（中国海外大厦） |
| 考试语言 | 可选中文繁体或英文 |
| 笔试时间 | 3月、9-10月 |
| 实操时间 | 1-2月、6月 |

### 考试通过率
- **警告**：据医学教育网信息，某大学培训班请外教上课，结果到香港考RN的通过率仅有**8%**
- 官方通过率数据：21%-41%（不同年份有波动）

### 薪资参考（2023-2024年数据）
| 职级 | 月薪（港币） |
|------|-------------|
| 登记护士(EN)起薪 | $20,540 |
| 登记护士顶薪(15年) | $43,610 |
| 注册护士(RN)起薪 | $32,545 |
| 注册护士顶薪(10年) | $52,370 |
| 护士长最高 | ~$75,000 |

### 案例1：知乎转述案例 - 通过优才计划赴港

**基本信息**
- **平台**：知乎
- **账号名**：年少有为Jeff
- **URL**：https://zhuanlan.zhihu.com/p/702986641
- **发布时间**：2024年6月12日
- **出国年份**：2024年
- **可信度标记**：[中介协助] / 转述朋友经历

**背景**
- 朋友通过优才计划获批，加上人才清单护士类30分
- 已顺利通过香港护士考试
- 目前在香港当注册护士

**时间线**
- 备考香港护士考试（投入大量时间精力）
- 申请优才计划（护士类人才清单加30分）
- 获批后赴港工作

**费用**
- 月薪从内地1万涨至香港4万+港币

**工作详情**
- 入职年假：18天（随年资逐年增加）
- 另有劳工假、公众假期
- 每周平均工作44小时
- 三班倒：
  - P下午班：14:00-22:00
  - A早上班：07:00-15:00
  - N通宵班：21:00-07:00

**卡点/挑战**
- 工作强度大，连续追班（今天P班明天A班，中间仅6小时休息）
- 每周一个通宵班
- 对身心健康压力大

**建议**
- "反正到哪里当护士都是做牛做马，那还不如选个薪资高的"
- 申请优才可以DIY，不需要中介

### 案例2：威普教育 - 2020年赴港护士

**基本信息**
- **平台**：威普爱生教育官网
- **URL**：https://www.wproedu.com/nurse/usrn/news/xmdt/show_1776.html
- **发布时间**：2023年8月4日
- **出国年份**：2020年
- **可信度标记**：[中介协助] / 培训学员案例

**背景**
- 从2020年开始在香港做护士
- 分享香港护士两种类型：登记护士(EN)和注册护士(RN)

**薪资详情**
- 登记护士(EN)：
  - 起薪：$20,540港币（政府总薪酬表第7级）
  - 第二年：$21,875港币（第8级）
  - 15年顶薪：$43,610港币（第21级）
- 注册护士(RN)：
  - 起薪：$32,545港币（第15级）
  - 10年顶薪：$52,370港币（第25级）

**工作体验**
- 年假：入职18天，5年到20天
- 劳工假和公众假期可调整休假日期
- 每月大约能放10天假期
- 工作时间：一更8小时，P班和A班轮流
- 偶尔连续上A更后再上N(通宵班)
- 大约一个月轮值一次通宵班

**卡点/挑战**
- P更下班后回家准备晚餐、洗澡，大约12点才入睡
- A更虽然7点上班，但需要6点起床，行程非常紧凑
- 班次转换对身体健康造成压力

**重要提醒**
- 成为香港注册护士**不需要**参加香港医院的实习
- 只需通过考试并提交相应的注册申请材料及费用即可
- 在香港工作满7年即可申请永居，拿香港身份

### 案例3：知乎中介文章 - 考试流程详解

**基本信息**
- **平台**：知乎专栏
- **账号名**：美国护士USRN（威普团队）
- **URL**：https://zhuanlan.zhihu.com/p/610791738
- **发布时间**：2023年3月2日
- **可信度标记**：[中介协助] / 营销内容

**考试资格**
- 年满21周岁
- 中专及以上护理学历
- 持有内地护士执照证书

**考试内容详解**
- **笔试部分**：
  - 满分200分，100分合格
  - 医学英语测试（甲卷）40分，必须取得合格成绩20分
  - 乙(Ⅰ)选择题60分：心理学、基础护理及身体系统护理学
  - 乙(Ⅱ)选择题+长答短答题100分：解剖生理学、营养学、微生物学、急救学、药理学等
- **实操部分**：
  - 时长1小时，包括3个部分
  - 健康评估（20分钟）
  - 给予药物（20分钟）
  - 无菌技术（20分钟）

**关键提醒**
- 必须在医学英语测验中取得合格成绩，否则全部成绩都将被判定为不及格
- 合格成绩：所有关键项目必须达到令人满意的表现 + 除去关键项目的剩余项目中至少达到50%的满意表现

### 案例4：安琪国际 - 大专学历赴港流程

**基本信息**
- **平台**：安琪国际护士培训官网
- **URL**：https://www.hnangel.com/news/1553.html
- **发布时间**：2025年11月20日
- **可信度标记**：[中介协助]

**申请条件**
- 学历：中专及以上护理相关学历
- 资质：持有有效期内的大陆护士执业资格证
- 年龄：RN需满21周岁，EN需满20周岁

**重要流程提醒**
- **提交渠道**：不支持网上提交！只能选择两种方式：
  1. 本人亲自到香港护士管理局现场递交
  2. 通过邮寄方式寄送材料
- **审核时间**：每年10月底前提交且材料齐全的申请，会统一审核
- **考试机会**：每人仅有**三次考试机会**，务必全力以赴，切勿抱着"试水"心态！

### 案例5：一亩三分地 - 通过香港考点考美国RN

**基本信息**
- **平台**：一亩三分地论坛
- **账号名**：have1look21
- **URL**：https://www.1point3acres.com/bbs/thread-994091-1-1.html
- **发布时间**：2023年5月12日
- **可信度标记**：[DIY成功] / 个人经历

**背景**
- 爱人在国内从事护士工作10余年
- 刚刚在香港的考点通过了美国注册护士执照考试（NCLEX-RN）

**重要信息**
- 香港可以作为NCLEX-RN的考点（对于想赴美工作的护士）
- 但通过后直接在美国找工作仍有身份障碍
- 医院一般不会为远在中国的人提供H1B签证

**网友回复精华**
- "签证是H1B，办签证的不多，办绿卡的比较多"
- "医院基本不会给提供HB1签证给远在中国的人，因为护士没绿卡很多医院根本不要"
- "只有一些难招人的养老院能会提供这个机会"
- "发达地区根本不缺护士，小地方缺也只是因为工资低"

## 三、失败/放弃案例

### 案例A：某大学培训班 - 低通过率警示

**基本信息**
- **来源**：医学教育网
- **URL**：https://www.med66.com/web/shewaihuli/ge1409256494.shtml
- **可信度标记**：[可疑营销] / 失败案例

**背景**
- 某大学培训班请了外教上课
- 结果到香港考RN的通过率仅有**8%**
- 4个通过RN考试的人，美国也不要
- 被推荐去了加拿大和澳洲做护理员
- 护士不愿意又被迫回国

**教训**
- 外教不了解中国护理与国外护理的差异
- "修改衣服"与"新做衣服"不一样
- 只有在中国接受过护理教育也在中国做过护理临床，在美国接受过护理教育也在美国做过的人，才了解真正的差异

## 四、黑中介诈骗案例（2024年大案）

### 案例B：2024年香港护工被骗大案

**基本信息**
- **来源**：江苏鑫正境外就业服务有限公司曝光平台
- **URL**：https://www.jsxzjw.com/detail-1623.html
- **发布时间**：2025年1月13日
- **案发时间**：2024年
- **可信度标记**：[黑中介诈骗] / 官方曝光

**案件规模**
- 涉及受害者：**3000多名护工**
- 涉案金额：**6000多万元人民币**
- 中介公司：启源（香港）顾问、珠海仁源劳务派遣公司等

**诈骗手法**
1. **虚假承诺**：大肆宣传赴港工作高薪，声称护工每月可轻松赚取1.5万港元以上
2. **伪造合作**：伪造与香港养老机构的合作关系
3. **层层抽佣**：采用类似传销手段，发展下线代理，每成功介绍一人可获得2000-4000元回佣
4. **拖延时间**：缴纳中介费后以各种理由拖延赴港时间，有的护工苦等逾一年仍未能成行

**受害者案例**
- **黎珊珊**：广西个人代理，曾在疫情期间申请赴港当护工失败，被高薪经历迷惑成为下线，一己之力在内地招聘了**482名护工**，中介费总额达**465万元**，却声称自己未得分文，全部上缴上线
- **南宁乐智康家政**：被珠海仁源劳务派遣公司骗走**2000多万元**，涉及近**1000人**
- **南宁珍心家政公司负责人余泽珍**：被骗走近百万元，**已是第三次被赴港劳务中介诈骗**

**警方处理**
- 部分涉案人员已被警方拘留
- 朱荣于2024年10月4日被广西南宁北湖派出所因涉嫌诈骗拘留
- 截至2024年12月，香港警务处已向内地相关部门转介共**14间**怀疑内地的"黑中介"资料

**警示**
- 在寻找工作机会时，务必保持警惕
- 仔细核实中介公司的资质和信誉
- 切勿轻信高薪承诺
- 相关部门需加强监管，严厉打击此类诈骗行为

## 五、关键卡点总结

### 考试阶段卡点
1. **三次机会限制**：每人仅有3次考试机会，不可轻视
2. **医学英语门槛**：笔试中医学英语测试必须合格，否则全部成绩作废
3. **实操考试难度**：健康评估、给药、无菌技术三部分都需合格
4. **通过率较低**：某培训班通过率仅8%，需认真备考

### 申请阶段卡点
1. **材料提交方式**：不支持网上提交，需本人赴港或邮寄
2. **审核时间节点**：每年10月底前提交，逾期将影响下一年考试
3. **仅限香港考点**：必须前往香港参加考试

### 工作阶段卡点
1. **工作强度大**：三班倒，连续追班，通宵班
2. **粤语要求**：虽然考试可选中文，但实际工作需要粤语
3. **需先通过考试才能申请工作签证**：不同于其他途径

### 中介风险
1. **虚假承诺高薪**：护工月薪1.5万港元以上的宣传
2. **卷款跑路**：6000万大案警示
3. **传销模式抽佣**：发展下线代理

## 六、建议与提醒

### 给DIY考生的建议
1. **充分准备**：不要抱着"试水"心态，三次机会宝贵
2. **医学英语**：提前准备医学英语测试
3. **官方教材**：优先使用香港护士管理局推荐的官方教材
4. **时间规划**：注意10月底的材料提交截止日期

### 给找中介者的建议
1. **核实资质**：查验中介是否有合法资质
2. **警惕高薪承诺**：月薪1.5万港元以上的护工宣传需警惕
3. **避免传销模式**：不参与发展下线代理的抽佣模式
4. **官方渠道优先**：尽量通过官方或正规渠道申请

### 特别警告
- 截至2024年12月，警方已转介14间怀疑黑中介资料给内地
- 3000人被骗6000万的案例警示：赴港务工需走正规渠道
