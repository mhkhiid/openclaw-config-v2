## 1. 从学历认证到拿身份的详细步骤

### 1.1 NNAS评估流程（第一步）

**什么是NNAS？**
National Nursing Assessment Service（国家护理评估服务）是加拿大国际护士学历认证的官方机构，负责评估国际护士的教育背景。

**两种服务模式对比：**

| 对比项 | 快速服务 (Expedited) | 常规服务 (Regular) |
|--------|---------------------|-------------------|
| 费用 | $750 CAD | $650 USD (~$845 CAD) |
| 处理时间 | 5个工作日 [NNAS官网] 2025-02 | 12周（约3个月） [NNAS官网] 2025-02 |
| 评估内容 | 国家层面护理教育评估 | 个体课程评估 |
| 多专业支持 | RN+LPN同时评估 | 需分别申请 |
| 额外监管机构 | 免费添加 | 同专业$55 USD，不同专业$220 USD |

**申请资格要求** [NNAS官网] 2025-02：
- 在加拿大境外完成护理高等教育
- 从未在加拿大获得护士执照
- 在境外有护士执照（仅快速服务要求）

**所需文件清单：**
1. 身份证明文件（护照等）- 申请人自行提交
2. 护理教育成绩单 - 由学校直接寄送
3. 护士执照证明 - 由发证机构直接寄送
4. 工作经历证明 - 由雇主直接寄送
5. 文件翻译（如需要）

**重要时间节点：**
- 申请有效期：12个月（从付费之日起）
- 评估报告有效期：12个月
- 过期申请重新激活费：$240 CAD（快速）/ $180 USD（常规）

### 1.2 各省注册流程

#### 安大略省 (CNO - College of Nurses of Ontario)

**流程步骤：**
1. 向NNAS申请评估
2. 收到NNAS Advisory Report
3. 向CNO提交注册申请
4. 完成注册考试（NCLEX-RN或RPN）
5. 通过Jurisprudence考试（法律伦理）
6. 满足语言要求
7. 获得临时注册 → 正式注册

**费用** [CNO官网参考]：
- 注册申请费：约$300-400 CAD
- 年度会员费：约$250-350 CAD
- NCLEX考试费：约$350 USD（Pearson VUE收取）

#### 不列颠哥伦比亚省 (BCCNM)

**特点：**
- 接受NNAS评估报告
- 要求通过NCLEX-RN
- 需提供英语能力证明

#### 阿尔伯塔省 (CNA - College of Nurses of Alberta)

**特点：**
- 要求2年临床经验
- 通过NCLEX-RN
- 可能需要补充课程

#### 魁北克省 (OIIQ - Ordre des infirmières et infirmiers du Québec)

**⚠️ Corner Case - 法语要求：**
- **强制法语要求**：魁北克是唯一强制要求法语的省份
- 需通过法语语言测试（如TEF Canada、TCF Canada）
- 或者完成魁北克认可的语言培训项目
- 注册护士考试用法语进行

**魁北克特殊流程：**
1. 向NNAS申请评估
2. 向OIIQ提交申请
3. 证明法语能力（B2水平）
4. 参加OIIQ组织的专业考试
5. 可能需要完成 bridging program

### 1.3 移民申请步骤

#### Express Entry 联邦快速通道

**适用类别：**
- Federal Skilled Worker (FSW) - 联邦技术工人
- Canadian Experience Class (CEC) - 加拿大经验类

**费用** [IRCC官网] 2025-02：
| 项目 | 费用 |
|------|------|
| 主申请人申请费 | $1,525 CAD [IRCC] |
| 配偶申请费 | $1,525 CAD [IRCC] |
| 受抚养子女 | $260 CAD/人 [IRCC] |
| 生物识别费 | $85 CAD/人（单人）/$170 CAD（家庭） |
| 体检费 | $200-400 CAD/人（因地区而异） |
| 语言考试费 | $300-350 CAD |

**处理时间：**
- 官方处理时间：6-8个月 [IRCC] 2025-02
- 实际等待时间：因CRS分数而异

#### 省提名项目 (PNP)

**安大略省 OINP (Ontario Immigrant Nominee Program)** [Ontario官网]：
- 医护人员优先类别
- 需先有安省雇主offer或已在安省工作
- 处理时间：约2-3个月省提名 + 6个月联邦处理

**各省PNP对比：**

| 省份 | 针对护士类别 | 是否需雇主offer | 处理时间 |
|------|-------------|----------------|---------|
| 安大略 | OINP雇主担保 | 是 | 2-3个月 [Ontario] |
| BC省 | BC PNP医疗类别 | 是 | 2-3个月 |
| 阿省 | AOS雇主担保 | 是 | 3-4个月 |
| 萨省 | SINP医疗类别 | 否（紧缺职业） | 2-3个月 |
| 曼省 | MPNP医疗类别 | 是 | 4-6个月 |

## 3. 时间线对比

### 3.1 官方处理时间 vs 论坛实际等待时间

| 阶段 | 官方时间 | 实际等待时间 | 备注 |
|------|---------|-------------|------|
| NNAS快速服务 | 5工作日 | 1-2周 | 文件齐全情况下 |
| NNAS常规服务 | 12周 | 3-5个月 | 可能因文件缺失延长 |
| NCLEX预约 | 即时-4周 | 2-6周 | 取决于考位availability |
| 省注册审批 | 4-12周 | 2-6个月 | 安省通常较慢 |
| EE联邦处理 | 6-8个月 | 6-12个月 | 取决于CRS分数 |
| EE省提名 | 2-3个月 | 2-4个月 | 安省OINP |

### 3.2 完整时间线示例

**最快路径（快速NNAS + 已有考位）：**
```
月份 0: 提交NNAS快速申请
月份 0.5: 获得NNAS评估报告
月份 1: 提交省注册申请 + 预约NCLEX
月份 2: 通过NCLEX
月份 3: 获得省注册执照
月份 3.5: 提交EE申请
月份 9-12: 获得PR身份
```
**总计：9-12个月**

**常规路径（常规NNAS）：**
```
月份 0: 提交NNAS常规申请
月份 3-4: 获得NNAS评估报告
月份 4: 提交省注册申请
月份 5-6: 通过NCLEX
月份 7-9: 获得省注册执照
月份 10: 提交EE申请
月份 16-20: 获得PR身份
```
**总计：16-20个月**

## 4. Corner Cases & 各省差异

### 4.1 魁北克特殊要求

**法语要求详情：**
- **B2水平**：魁北克政府要求移民申请人证明法语口语达到B2水平
- **考试选项**：
  - TEF Canada（法语水平测试）
  - TCF Canada
  - DELF/DALF
- **豁免情况**：
  - 在魁北克完成护理教育可豁免
  - 有魁北克雇主offer可部分豁免

**魁北克注册流程差异：**
1. 除NNAS外，可能需要额外的学历认证
2. 注册考试用法语进行
3. 可能需要完成"课程对比评估"
4. 移民走魁北克技术移民（QSWP），不与EE互通

### 4.2 其他各省差异

| 省份 | 特殊要求 | 备注 |
|------|---------|------|
| BC省 | 需提供无犯罪记录 | 所有外国国家居住超过6个月的 |
| 阿省 | 2年工作经验要求 | 可毕业后累积 |
| 曼省 | 紧缺职业优先 | 护士属紧缺职业 |
| 萨省 | 有紧缺职业直通类别 | 无需雇主offer |

### 4.3 常见延误原因

1. **文件不全**：NNAS申请常因文件缺失被延迟
2. **学校/雇主配合**：需第三方直接寄送文件，耗时较长
3. **翻译问题**：非英语/法语文件需认证翻译
4. **考位紧张**：NCLEX考位可能需要提前数月预约
5. **背景调查**：部分国家背景调查耗时较长
