### 步骤1-3：准备阶段

#### 步骤1：语言考试

**雅思 (IELTS)**
- **报名链接**：https://www.ielts.org/zh-cn
- **考试类型**：雅思学术类 (Academic)
- **分数要求**：
  - 大多数州要求：总分6.5，单项不低于6.0
  - 部分州（如纽约）：总分7.0
  - CGFNS要求：总分6.5，口语7.0（某些情况）
- **考试费用**：约¥2,170（中国大陆）
- **成绩有效期**：2年

**托福 (TOEFL iBT)**
- **报名链接**：https://www.ets.org/toefl
- **分数要求**：
  - 总分83分（部分州要求更高）
  - 阅读/听力：20分，口语/写作：20分
- **考试费用**：$245 USD
- **成绩有效期**：2年

**注意**：
- 雅思在中国大陆考点较多，推荐优先选择
- 成绩需直接由考试机构发送至CGFNS（机构代码在申请时提供）
- 语言成绩有效期为2年，规划好时间避免过期

#### 步骤2：州选择研究

**为什么州选择很重要：**
不同州对外国护士的要求差异很大，影响整个认证流程。

**热门州对比：**

| 州 | 优势 | 特殊要求 | CGFNS要求 |
|----|------|----------|-----------|
| **纽约州 (NY)** | 不要求美国社保号 | 需CVS-NY认证 | CGFNS Credential Verification |
| **加州 (CA)** | 华人社区大 | 要求额外课程 | CGFNS Certificate或CES |
| **德克萨斯 (TX)** | 处理速度快 | 需指纹背景调查 | CES Professional Report |
| **佛罗里达 (FL)** | 无州税 | 严格的执照验证 | CGFNS Certificate |
| **伊利诺伊 (IL)** | 芝加哥医疗资源丰富 | 要求英语成绩 | CES |

**官方资源：**
- **NCSBN州列表**：https://www.ncsbn.org/contact-bon.htm
- **各州护理委员会网站**：在NCSBN网站找到对应州链接

**建议**：
1. 先选择一个州开始申请（推荐纽约州，因流程相对简单）
2. 获得执照后可通过背书(Endorsement)转到其他州
3. 考虑该州是否有排期优势（某些州PERM处理更快）

#### 步骤3：学历文件准备

**CGFNS要求的文件清单：**

| 文件类型 | 要求 | 备注 |
|----------|------|------|
| **护理毕业证** | 原件或公证件 | 需学校直接寄送 |
| **护理成绩单** | 详细课程描述 | 需学校密封邮寄 |
| **护士执照** | 当前有效执照 | 需发证机构直接验证 |
| **高中毕业证** | 部分情况需要 | 作为前置学历证明 |
| **身份证明** | 护照复印件 | 申请时上传 |

**文件要求详情：**

1. **护理学校成绩单必须包含**：
   - 理论学时和临床实习学时
   - 课程描述（儿科、妇产科、精神科、成人护理等）
   - 成绩评分标准说明

2. **翻译要求**：
   - 必须由专业翻译机构翻译
   - 翻译件需附翻译员声明
   - 部分文件需公证

3. **邮寄要求**：
   - 学校必须直接邮寄至CGFNS（不能经申请人手）
   - 使用学校官方信封并密封盖章
   - 推荐使用DHL/FedEx可追踪快递

**官方文件清单PDF**：
- 可在CGFNS账户内下载完整文件清单
- 或直接联系：support@cgfns.org

#### 步骤6：VisaScreen®申请

**什么是VisaScreen：**
VisaScreen®是CGFNS提供的签证筛查服务，是申请EB-3移民签证或H-1B工作签证的必需文件。

**申请费用：**
| 项目 | 费用 |
|------|------|
| VisaScreen申请费 | $545 |
| I-140支持信（如需要） | $225 |
| 额外报告副本 | $95/份 |
| 加急服务 | $595 |

**官方链接**：https://www.cgfns.org/services/visascreen/

**申请前提条件：**
1. 通过NCLEX-RN考试（或持有美国州执照）
2. 英语成绩达标（雅思/托福）
3. 完成CGFNS学历评估

**申请步骤：**

1. **在线申请**
   - 登录CGFNS账户
   - 选择"VisaScreen® Service"
   - 支付$545申请费

2. **提交材料**
   - 英语成绩（直接由考试机构发送）
   - NCLEX成绩报告
   - 当前护士执照验证

3. **审核过程**
   - CGFNS审核所有材料
   - 如有问题会通过邮件联系

4. **获取证书**
   - 审核通过后发放VisaScreen®证书
   - 证书有效期：5年
   - 可选择邮寄或电子发送给移民局

**处理时间：**
- [官方] 2-4个月
- [实际经验] 3-6个月
- 加急服务：1-2周（额外$595）

### 步骤7-8：考试阶段

#### 步骤7：NCLEX-RN考试注册

**NCLEX简介：**
NCLEX-RN（National Council Licensure Examination for Registered Nurses）是美国注册护士执照考试，所有州都要求通过此考试。

**注册流程：**

**第一步：向州护理委员会申请考试资格**

1. 访问目标州护理委员会网站
2. 提交RN licensure by examination申请
3. 支付州申请费（各州不同，$50-300不等）
4. 州委员会审核CGFNS报告

**第二步：在Pearson VUE注册**

- **官方网站**：https://www.pearsonvue.com/nclex/
- **注册步骤**：
  1. 创建Pearson VUE账户
  2. 支付考试费$200
  3. 等待收到Authorization to Test (ATT)

**第三步：获取ATT**

- ATT通过邮件发送
- 包含考试授权号码和有效期（通常90天）
- 必须在有效期内预约考试

**NCLEX考试费用：**
| 项目 | 费用 |
|------|------|
| NCLEX考试费 | $200 |
| 州申请费（各州不同） | $50-300 |
| 国际考点额外费 | 无（香港考点） |

**官方资源：**
- NCSBN NCLEX信息：https://www.ncsbn.org/nclex.htm
- Pearson VUE注册：https://www.pearsonvue.com/nclex/

#### 步骤8：预约考试&参加NCLEX

**香港考点信息：**

| 项目 | 详情 |
|------|------|
| **地址** | Pearson Professional Centers, Hong Kong |
| **具体位置** | 香港岛/九龙（具体地址在预约时确认） |
| **考点查询** | https://wsr.pearsonvue.com/testtaker/profile/Signup.htm |
| **考位情况** | 通常需提前2-4周预约 |

**预约步骤：**

1. 收到ATT后登录Pearson VUE账户
2. 选择"Schedule Exam"
3. 选择考试中心（搜索"Hong Kong"）
4. 选择可用日期和时间
5. 确认预约并保存确认信

**考试当天：**
- 携带有效护照
- 提前30分钟到达
- 不能携带任何个人物品进入考场
- 考试时长：最长5小时（含休息时间）

**成绩获取：**
- 考试结束后48小时可通过NCSBN网站付费查询快速结果
- 正式结果由州委员会通知
- 通过分数线：使用计算机自适应测试(CAT)，题目数量75-145题

**重考政策：**
- 如未通过，需等待45天才能重考
- 需重新申请ATT并支付考试费
- 每年最多考8次

### 步骤9-10：移民阶段

#### 步骤9：雇主匹配&I-140申请

**雇主匹配渠道：**

| 渠道 | 说明 | 链接 |
|------|------|------|
| **CGFNS Alliance** | CGFNS官方雇主匹配 | https://www.cgfns.org/services/alliance/ |
| **Avant Healthcare** | 大型护士招聘机构 | https://www.avanthealthcare.com/ |
| **O'Grady Peyton** | 国际护士招聘 | https://www.ogradypeyton.com/ |
| **直接申请医院** | 大型医院系统 | HCA, Kaiser Permanente等官网 |
| **护士招聘网站** | Indeed, LinkedIn | 搜索"RN sponsorship" |

**雇主筛选要点：**
1. 确认雇主有EB-3移民担保经验
2. 确认雇主愿意支付PERM和I-140费用
3. 了解合同条款（工作年限、违约金等）
4. 确认薪资和福利符合当地标准

**I-140申请（Immigrant Petition for Alien Workers）：**

| 项目 | 详情 |
|------|------|
| **表格** | I-140 |
| **费用** | $715（雇主支付） |
| **加急费（可选）** | $2,805（15天内出结果） |
| **官方链接** | https://www.uscis.gov/i-140 |

**I-140申请流程：**

1. **PERM劳工证**（由雇主完成，先于此步骤）
   - 招聘广告期：约2-3个月
   - DOL处理时间：6-12个月
   
2. **准备I-140材料**
   - 批准的PERM劳工证
   - 护士学历证明
   - 护士执照
   - 雇主财务证明
   - VisaScreen证书

3. **提交I-140申请**
   - 雇主或律师提交至USCIS
   - 可选择加急处理

4. **等待审批**
   - 常规处理：6-12个月
   - 加急处理：15个日历日

**表格下载**：
- I-140表格：https://www.uscis.gov/sites/default/files/document/forms/i-140.pdf
- 说明文件：https://www.uscis.gov/i-140

#### 步骤10：I-485调整身份或领事处理

**排期等待跟踪：**

**签证排期查询：**
- **官方来源**：美国国务院签证公告
- **链接**：https://travel.state.gov/content/travel/en/legal/visa-law0/visa-bulletin.html
- **护士类别**：EB-3（第三优先）

**排期跟踪方法：**
1. 每月10日左右查看新一期签证公告
2. 关注"Final Action Dates"（最终行动日期）
3. 护士属于"Professionals"类别

**当前排期情况（2025-2026参考）：**
- 中国大陆出生：约2-4年
- 其他国家/地区：通常当前无排期

**身份调整途径：**

| 途径 | 适用情况 | 费用 |
|------|----------|------|
| **I-485调整身份** | 已在美国合法居留 | $1,225 + $85指纹 |
| **领事处理** | 在美国境外 | DS-260费用：$345 + 体检费 |

**I-485调整身份流程：**

1. **申请条件**
   - I-140已批准
   - 排期已到（如适用）
   - 在美国有合法身份

2. **提交材料**
   - I-485表格
   - I-693体检报告
   - I-864经济担保（如需要）
   - 护照、签证、入境记录

3. **生物识别**
   - 收到打指纹通知
   - 前往ASC中心采集指纹

4. **面试（如需要）**
   - 部分申请人需面试
   - 携带所有原件

5. **获得绿卡**
   - 批准后会先收到I-551印章
   - 绿卡邮寄到家

**官方资源：**
- I-485表格：https://www.uscis.gov/sites/default/files/document/forms/i-485.pdf
- 说明文件：https://www.uscis.gov/i-485

**处理时间：**
- [官方] 12-24个月
- [实际经验] 18-36个月（因地区而异）

## ⏱️ 时间总表

### 官方处理时间 vs 实际经验

| 步骤 | [官方说法] | [实际经验] | 可并行？ |
|------|------------|------------|----------|
| 1. 语言考试准备 | 自定 | 1-3个月 | ✅ |
| 2. 州选择研究 | 1-2周 | 1-2周 | ✅ |
| 3. 学历文件准备 | 2-4周 | 1-2月 | ✅ |
| 4. CGFNS评估 | 3-6月 | 4-8月 | ⚠️ 需材料齐全 |
| 5. 材料邮寄审核 | 包含在4 | 包含在4 | ⚠️ 与4同时 |
| 6. VisaScreen | 2-4月 | 3-6月 | ✅ 可与NCLEX并行 |
| 7. NCLEX注册/考试 | 2-3月 | 3-6月 | ✅ |
| 8. 雇主匹配 | 不定 | 1-6月 | ⚠️ 需在通过NCLEX后 |
| 9. PERM劳工证 | 6-12月 | 8-18月 | ❌ 必须串行 |
| 10. I-140申请 | 6-12月 | 8-18月 | ❌ 需PERM批准 |
| 11. 排期等待 | 2-4年 | 2-5年 | ⚠️ 中国大陆出生 |
| 12. I-485/签证 | 12-24月 | 18-36月 | ❌ 需排期到+I-140批 |

### 并行 vs 串行说明

**可以并行的步骤：**
```
阶段1-3：准备阶段（全部可并行）
├── 学习雅思/托福
├── 研究目标州
└── 准备学历文件

阶段4-7：认证考试阶段（部分可并行）
├── CGFNS评估（提交后等待）
├── 准备NCLEX（可同时学习）
└── 预约NCLEX考试（收到ATT后）
```

**必须串行的步骤：**
```
阶段8-12：移民阶段（必须串行）
├── 通过NCLEX → 找到雇主
├── 雇主 → 提交PERM
├── PERM批准 → 提交I-140
├── I-140批准 + 排期到 → I-485/签证
└── I-485批准 → 获得绿卡
```

### 总时间估算

| 路径 | 最短 | 正常 | 最长 | 说明 |
|------|------|------|------|------|
| **无排期国家** | 2年 | 3年 | 4年 | 非中国大陆/印度/菲律宾 |
| **中国大陆排期** | 4年 | 5-6年 | 7年+ | 当前EB-3排期情况 |
| **加急处理** | -6月 | -1年 | -1.5年 | 使用PP加急服务 |

**关键时间节点：**
- **第1年**：完成CGFNS、NCLEX考试
- **第2-3年**：找到雇主、PERM、I-140
- **第3-5年**：排期等待（中国大陆）
- **第5-6年**：I-485批准，获得绿卡
