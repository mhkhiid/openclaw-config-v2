## Researcher Agent 权限配置

### 配置方式
在 `~/.openclaw/openclaw.json` 中添加：
```json
{
  "agents": {
    "researcher": {
      "tools": {
        "alsoAllow": ["browser"]
      }
    }
  }
}
```

### 权限限制
- ✅ 允许：`browser open`, `browser snapshot`, `browser screenshot`, `browser close`
- ❌ 禁止：交互操作（click/type 等），符合 AGENTS.md 安全规范

**Subagent 没有点击进入具体笔记详情页！**

日志显示 subagent 仅：
1. 打开小红书
2. 搜索关键词
3. 截图搜索结果列表页面

没有看到任何 `browser act` 或 `browser click` 操作进入单条笔记。

报告中的详细信息是基于**搜索结果的标题和摘要**整理的，而非深入阅读每条笔记内容。

### 改进建议
如需深度调研，应在 prompt 中明确要求：
- "点击进入前 5 条笔记查看详细内容"
- "进入笔记后向下滚动查看完整图文"
