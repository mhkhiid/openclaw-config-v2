## ✅ 二、选口红的核心原则

### 🎨 1. 色调选择原则

**安全色系（闭眼入）：**
- 🔴 **正红色** - 迪奥#999、YSL#1 - 经典不出错，显白有气场
- 🍅 **烂番茄色** - 阿玛尼#405 - "黄皮显白天菜"，不挑肤色
- 🍒 **车厘子色/浆果色** - 显白有活力，各年龄段都能驾驭
- 🍁 **红棕色/枫叶色** - 秋冬首选，温柔又高级

**选色公式：**
```
✓ 带"fire"、"flame"字样的色号 = 显白、百搭
✓ 正红、番茄红、豆沙红 = 亚洲黄皮友好
✓ 土橘系、南瓜色、赤茶色 = 潜力潮流色
✗ 荧光粉、亮紫、裸灰 = 雷区
```

### 💧 2. 质地选择

| 质地类型 | 特点 | 适合人群 |
|---------|------|---------|
| **哑光** | 高级显色、持久 | 唇部状态好、喜欢欧美妆 |
| **滋润** | 保湿不干、光泽感 | 唇部干燥、日常通勤 |
| **丝绒** | 介于两者之间 | 百搭选择，大部分人都适合 |
| **镜面/水光** | 玻璃唇效果 | 年轻、喜欢日韩妆 |

**建议：** 不确定就选**丝绒质地**或**滋润质地**，出错率低。

### 🏷️ 3. 品牌推荐

**大牌保险款（送礼有面子）：**
- 迪奥 Dior - #999 正红色，经典中的经典
- 阿玛尼 Armani - #405 烂番茄，"黄皮亲妈"
- 纪梵希 Givenchy - #N37 红丝绒，显白天花板
- 香奈儿 Chanel - #627 山茶花红棕
- TF Tom Ford - #27 水红粉色，谁涂都好看

**性价比之选：**
- 稚优泉、完美日记、花西子（适合学生党）

## 💝 三、送礼的加分细节

### 🎁 1. 包装加分

**必须做到：**
- ✓ 专柜礼盒装 > 单支裸装
- ✓ 品牌官方礼袋 + 丝带
- ✓ 避免用透明塑料袋直接装

**进阶操作：**
- ✨ 金属质感包装纸 - "指尖刚碰到就微微一顿的质感"
- ✨ 定制礼盒封面 - 印上对方名字或祝福语
- ✨ 搭配同品牌其他小样（香水、护肤品）

### ✍️ 2. 刻字服务

**很多品牌支持口红管身刻字：**
- 刻对方名字/昵称（如：To 小美）
- 刻纪念日（如：2024.2.14）
- 刻简短情话（如：Love U、You are beautiful）

**注意：** 刻字需要提前咨询专柜或官网，部分品牌仅限特定产品。

### 💌 3. 贺卡与仪式感

**贺卡内容建议：**
```
简短版："这支颜色我查了很久，觉得特别衬你。"
走心版："希望你涂上它的时候，能想起我。"
俏皮版："口红我买了，但只准涂给我看😏"
```

**仪式感加分项：**
- 🕯️ 搭配香薰蜡烛，营造浪漫氛围
- 🌹 配合鲜花一起送（如永生花+口红套装）
- 📸 录一段小视频说明选色心路历程

## 📝 四、终极避坑检查清单

送礼物前，先问自己：

- [ ] 我确定她的肤色是冷皮还是暖皮吗？
- [ ] 她平时喜欢淡妆还是浓妆？
- [ ] 她有没有类似的色号了？（回家偷偷看看她的化妆台）
- [ ] 我选的是不是"安全色系"（正红/番茄红/豆沙红）？
- [ ] 我有没有咨询过其他女性朋友？
- [ ] 包装是否足够精致？
- [ ] 是否准备了贺卡/手写卡片？
