# Alex 数字孪生知识库

> 创建时间：2026-02-14
> 维护者：助手圆
> 目的：系统化记录和提炼Alex的认知、判断、习惯和价值观

## 架构说明

本知识库是Alex数字孪生的核心资产。与USER.md的关系：
- **USER.md** = 精华摘要（每次session自动加载）
- **digital-twin/** = 完整知识库（按需深度读取）

USER.md中的关键信息应从本库蒸馏而来。

## 目录结构

| 目录 | 内容 | 更新频率 |
|------|------|---------|
| `worldview/` | 世界观、价值观、人生哲学 | 低频（重大认知变化时） |
| `cognition/` | 决策模式、思维风格、盲区 | 中频（持续观察积累） |
| `business/` | 行业洞察、商业模式、市场判断 | 中频 |
| `communication/` | 语言习惯、表达模式、沟通风格 | 中频（从日常对话提炼） |
| `life-flow/` | 日常习惯、偏好、情绪模式 | 中频 |
| `relationships/` | 人际网络、关键关系 | 低频 |
| `distilled/` | 周度/月度蒸馏提炼 | 每周 |

## 数据来源

1. **日常对话自然提炼**（主要来源）
2. **孪生夜话**（每晚9-11点，~10分钟深度对话）
3. **每周观察报告**（周末生成review）
4. **被动验证**（实际任务中的偏差修正）

## 使用规则

- 记录事实和观察，不做过度推断
- 标注信息来源和日期
- 标注置信度（高/中/低）
- 每周进行一次蒸馏提炼
- 发现与已有记录矛盾时，记录矛盾而非直接覆盖
