# 微演绎格式指南
> 适用：通过换衣服一人饰演多个角色的对比/冲突类视频

---

## 脚本格式
对话体剧本，包含：角色设定 + 服装标记 + 场景转换指令 + 台词
格式示例：
```
[角色设定]
角色A：国内护士（白大褂/便装）
角色B：德国护士（蓝色工服）
[场景1 | 分屏/切换]
角色A（表情/状态）："台词..."
[切换 → 角色B]
角色B（表情/状态）："台词..."
[结尾字幕]
左：总结A  右：总结B
```

## 节奏规则
- 台词节奏偏快，有戏剧感
- 最优时长：**35-50秒**（微演绎宜短不宜长）
- 每个角色单次台词不超过3句
- 全片角色切换4-8次
- 快切=节奏感，慢切=拖沓

## 角色设计原则
- 最多2-3个角色（换装成本限制）
- 常用对比组：
| 对比组 | 适合选题 |
|--------|----------|
| 国内护士 vs 德国护士 | 权益类、夜班类、中德差异 |
| 护士 vs 护士长 | 护士长类、权益类 |
| 新手护士 vs 老护士 | 出国路径、职业发展 |
| 护士 vs 病人 | 事件类 |
| 过去的我 vs 现在的我 | 个人故事（慎用） |
- 视觉区分要明显：衣服颜色/配饰/发型/表情

## 钩子适配
- 钩子 = 第一个场景的前1-2句台词
- 最有效方式：**直接进入冲突场景**，不做铺垫
- 口播版："护士被病人骂肯定不容忍"
  → 微演绎版：
  [病人 | 坐床上 | 指镜头] "你们护士怎么回事！"
  [切换 → 护士 | 门口 | 冷静] "在德国 这种病人直接开出院"

## 结构适配
| 叙事结构 | 微演绎适配度 | 说明 |
|----------|-------------|------|
| A 反常识→验证 | ⭐⭐ | 可以但不是最佳形式 |
| B 极端场景→制度 | ⭐⭐⭐⭐ | 冲突场景很适合演 |
| C 日常Vlog | ⭐ | 不适合 |
| D 痛点→逆袭 | ⭐⭐ | 可以用"过去vs现在"但削弱真实感 |
| **对比结构（专属）** | ⭐⭐⭐⭐⭐ | 微演绎天然最强结构 |

微演绎专属对比结构：
```
国内场景A → 德国场景B → 国内场景A' → 德国场景B' → 字幕总结
```

## 选题适配表
| 选题类型 | 适配度 | 说明 |
|----------|--------|------|
| 权益/制度类 | ⭐⭐⭐ | 中德对比天然适合 |
| 夜班类 | ⭐⭐ | 可做"国内夜班vs德国夜班" |
| 病人事件类 | ⭐⭐⭐ | 护士vs病人冲突戏剧性强 |
| 护士长类 | ⭐⭐⭐ | 护士vs护士长有张力 |
| 个人故事类 | ⭐ | 不建议，演绎削弱真实感 |
| 工资类 | ⭐⭐ | 可做但不如口播直接 |
| 招聘类 | ⭐ | 不建议 |

## DO/DON'T
- ✅ 每次切换用明显视觉/声音断点（转场音效/闪切）
- ✅ 台词允许比口播更戏剧化、夸张化
- ✅ 结尾用字幕做数据对比总结（感性演绎→理性冲击的收尾）
- ✅ 国内角色可以略带夸张但保持尊重
- ❌ 不要超过3个角色
- ❌ 不要超过50秒（演技和节奏撑不住）
- ❌ 不要用微演绎讲信息密集型内容（出国流程等）
- ❌ 不要过度丑化"国内护士"角色（引发争议）
- ❌ 不要让换装过程出现在画面中
