# 钩子公式库
> 最后更新：2025-11-25
> 用途：各账号生成钩子时的公式参考+历史案例

---

## 通用原则
前5秒必须包含以下至少2个要素：
1. **情绪词**：真服了 / 震惊了 / 你敢信吗 / 太离谱了 / 直接爆炸了 / 无语了
2. **具体数字**：2000块 / 28张床 / 21个护士 / 56天 / 8个星期 / 3414欧
3. **反常识**：加班违法 / 夜班要抢 / 不休假挨骂 / 怀孕直接休 / 催发小费
4. **冲突场景**：被病人骂 / 全科请假 / 护士投诉护士长

## 模型A：反常识炸弹（历史最强，百万级视频中占62%）
### 公式
```
"在德国 [一个国内习以为常的行为] 居然是 [违法的/被罚的/被禁止的]"
```
或
```
"[情绪词] [德国人/德国医院]的[脑回路/规定] [具体反常识事实]"
```

### 可填入的关键词组合
| 国内习以为常的行为 | 德国的反转结果 |
|-------------------|---------------|
| 护士擅自加班 | 是违法的 |
| 带病坚持上班 | 被同事赶回家 |
| 下夜班去开会 | 根本不存在 |
| 连续工作8小时不休息 | 医院要被罚钱 |
| 护士长不批病假 | 护士长没有这个权力 |
| 让护士免费加班 | 医院会被告 |
| 一个护士管40个病人 | 医院会被处罚 |
| 夜班被迫上 | 夜班要靠抢 |
| 护士收到小费不敢要 | 理直气壮去催 |

## 各选题类型的钩子适配说明
### 权益/加班类
- 首选：模型A（反常识）
- 必须包含"违法""罚钱""不允许"等强冲击词
- 历史最强组合：反常识 + 违法关键词

### 夜班类
- 首选：模型A（反转型）或模型B（数字冲击）
- 反转核心："夜班不是被安排的 而是靠抢的"
- 数字核心："一个夜班2000块"

### 休假/病假类
- 首选：模型B（数字冲击）或模型D（冲突事件）
- 数字核心：30天/8个星期/6周
- 事件核心：全科请假/护士长催休

### 护士证/出国路径类
- 首选：模型C（共情痛点）
- 必须触及身份焦虑："丫鬟证""找不到工作"
- 转折核心：护士证的隐藏价值

### 个人故事类
- 首选：模型C（共情痛点）
- 必须暴露弱点/缺陷：中专、158cm、农村
- 核心句式：不是不想回 而是...

### 工资/收入类
- 首选：模型B（数字冲击）
- 必须出现税后/到手数字
- 换算为人民币

### 互换比/科室管理类
- 首选：模型B（数字冲击）
- 核心数据：28床/21护士

### 中德差异类
- 首选：模型A或模型F（假设反推）
- 建议单点深挖而非清单罗列

### 护士长/护理部类
- 首选：模型A（角色反转类反常识）
- 核心："催你下班/求你休假/替你撑腰的护士长"

### 招聘/学员类
- 首选：模型C（身份逆袭）的简化版
- 重点突出学员背景（学历/年龄）引发代入感
- 时长必须≤35秒
