# 全局创作规则
> 最后更新：2025-11-25
> 用途：所有账号、所有选题、所有格式通用的硬性规则

---

## 一、铁律（违反即废稿）
1. **前5秒不得出现自我介绍**（"大家好我是XX"）
2. **前30秒不得出现品牌名/公司名**（"海狸未来"等）
3. **不得出现"关注我""点赞""评论区见"等平台限流话术**
4. **视频时长不得超过85秒**（历史数据：>85秒的视频0条百万爆款）
5. **同一钩子句式不得在3个月内重复使用第3次**（5s留存断崖下降>10%）
6. **同一主题最短复用间隔：90天**
7. **每月招聘/学员类内容不得超过2条**

---

## 二、强制包含项
1. 每条脚本包含 **3-5个数据支柱**（参见data_pillars.md）
2. 前5秒包含 **≥2个冲击要素**（情绪词/数字/反常识/冲突场景）
3. **30秒处必须有第二信息炸弹**（防止中途划走）
4. 结尾必须有 **互动钩子**（提问/选择/感慨）

---

## 三、语言风格规范
### 必须
- 口语化，像跟朋友聊天
- 短句为主，每句10-15字内断句
- 第一人称叙述
- 用具体数字代替模糊描述（"2000块"不说"很多钱"）

### 禁止
- 书面语/文绉绉的表达
- "首先…其次…最后…"等列举句式
- "众所周知""不可否认"等论文用语
- 过度使用感叹号

---

## 四、内容红线
1. **不直接贬低国内**——说"在德国是这样的"，让观众自己对比
2. **不承诺绝对收入**——"起薪3414欧"而非"保证月入3万"
3. **不回避德国的缺点**——"存不了大钱""离家远""文化差异"适度提及
4. **不丑化"国内护士"角色**（微演绎中尤其注意）
5. **不把病假描述为"摸鱼/骗病假"**（措辞用"合法权益"）

---

## 五、两条爆款路径
### 路径1：分享驱动型（破圈用）
- 特征：分享率 >2.5%，话题辐射面超出护士群体
- 适用选题：加班违法、护士证价值、年假排班
- 关键：让所有打工人都能共鸣
- 占比建议：30%

### 路径2：算法驱动型（培粘性）
- 特征：完播率 >20%，5s留存 >55%
- 适用选题：夜班揭秘、为什么不回国、Vlog日常
- 关键：内容本身极度吸引，看完不划走
- 占比建议：40%

### 日常/转化型
- 占比建议：30%

---

## 六、内容复用规则
| 条件 | 要求 |
|------|------|
| 间隔时间 | ≥90天 |
| 钩子 | 必须更换句式和角度 |
| 核心数据 | 可保留 |
| 主体内容 | 调整≥30% |
| 连续复用 | 同一主题最多复用3次，之后需寻找全新角度 |

---

## 七、质量审核检查清单
每条脚本发布前逐项勾选：
- [ ] 前5秒含≥2个冲击要素
- [ ] 总时长在目标区间内
- [ ] 包含3-5个数据支柱
- [ ] 30秒处有第二信息点
- [ ] 无DON'T清单违规项
- [ ] 结尾有互动钩子
- [ ] 无过度推销/广告感
- [ ] 语言口语化自然（不像AI写的）
- [ ] 无重复啰嗦段落
- [ ] 关键数字准确（核对data_pillars.md）

评分标准：每项10分，满分100分
- ≥80分 → 通过
- 60-79分 → 修改后重审
- <60分 → 废弃重写
