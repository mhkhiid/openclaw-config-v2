# 跨账号经验迁移记录
> 用途：当某个账号验证了有效的新策略，记录在此供其他账号参考
> Orchestrator负责写入和广播

---

（初始状态：暂无跨账号数据。以下为结构模板）

## 模板
```
## YYYY-MM-DD
- **来源账号：** Account X
- **发现：** [具体描述]
- **数据支撑：** [关键指标]
- **建议适用范围：** [哪些账号/哪些选题/哪些格式]
- **已应用于：** ☐ Account A  ☐ Account B  ☐ Account C
- **验证结果：** [待验证 / 已验证有效 / 已验证无效]
```

---

## 已知的平台差异（可跨账号参考）
### 抖音
- 最优口播时长：45-57秒
- 发布最佳时段：17:00-18:30（主力）/ 9:00-11:00（被低估）
- 字幕必须有（平台推荐）

### 小红书（如有账号）
- Vlog不超过60秒（完播率断崖）
- 图文笔记可以补充视频覆盖不到的信息密集型内容
- 封面图决定点击率

### 视频号（如有账号）
- 中老年用户占比高
- 分享到微信群/朋友圈是主要增长路径
- 情感共鸣类（个人故事/为什么不回国）可能比年轻平台更有效
